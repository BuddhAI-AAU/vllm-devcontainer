BuddhAI is a custom conversation pipeline build with sovereignty in mind. Self reliance and data privacy are prioritized, which sets this project apart from many others. Everything is hosted and maintained locally at AAU and every prompt written by the user and generated by the model of choice are strictly restricted to be viewed only by the user themselves.

Firstly, we believe that this promotes trust and confidence, as the user will feel less inclined have negative emotions regarding asking "stupid questions", as there are no one to judge them.

Secondly, we believe that this would give an incentive to use our services over other commercially available ones, such as co-pilot and chat-GPT. We are transparent with how the user's data is processed, what we store, where and for how long and have the philosophy that data should only be accessed by another party if requested and then consented to by the user. Not the practice of the user consenting to their data being viewed and used before they even start to use the service, which seems to be common practice many places.
